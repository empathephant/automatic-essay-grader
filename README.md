# automatic-essay-grader
Machine learning project that attempts to grade essays written by non-native English speakers on a scale from 1 - 7.

## Dependencies

* nltk
* scipy
* numpy
* pandas
* matplotlib
* sklearn

## Set-up

Run the file "machinelearn.py"
